Области видимости, нюансы

В index.html мы подключили также два JavaScript файла. В file-1.js, без предварительного объявления, запишем в переменную boss (из глобальной области видимости) строковое значение

        boss = 'Keks';

В file-2.js, который подключён самым последним, выведем в консоль значение переменной boss:

        console.log(boss);

Открыв страницу index.html в браузере, мы увидим в консоли значение Keks!

Это возможно, благодаря глобальной области видимости. Когда мы объявляем переменные в теге script, они все записываются в глобальную область видимости, которой является объект window.
